开设课程

发布时间： 2017-04-26

实操类

  • 1.基础班：会计基础、出纳、商业
  • 2. 新秀班：出纳、商业、工业、店铺业、财务软件、新手上路技巧、小型企业网上申报
  • 3. 精英班：新秀班、税务实务、工商一体化、装饰业、连锁酒店业、财务管理基础、常规企业网上申报
  • 4. 主管班：精英班、E表、六大管理课、所得税年末汇算清缴及其他涉税业务
  • 4. 护航班：主管班、六大财税管控课程、财务核算实验室、税务操作实验室

证书类

  • 1. 初级职称：初级基础班、初级精讲班、初级真题班
  • 2. 中级职称：中级基础班、中级精讲班、中级真题班
  • 3. 高级职称：高级基础班、高级精讲班、高级真题班
  • 4. 注册会计师：会计、审计、财务成本管理、税法、经济法、战略与风险管理

课程收益：

    新秀班

  • 1、亲自动手填制各种票据，如涉及资金的（支票，进账单，缴款单，电汇单，网银单，承兑汇票），涉及货物进出的入库单，调拨单，领料单，销售出库单，发票等
  • 2、企业如何从无到有？筹建期如何做账？转入正式经营如何做账？
  • 3、工业企业订单式成本核算？
  • 4、如何利用财务软件建账出表？
  • 5、通晓财务软件安装、新建多行业账套、凭证录入、审核、会计报表出具、查询明细总分类账及年度结转。
  •  

    • 精英班

  • 1、货物进大型商超如何结算？
  • 2、专卖店里的钱货如何做账？
  • 3、企业里涉及的税种主要有哪些？计税依据是什么？
  • 4、日常账务中如何重点关注视同销售的业务处理？
  • 5、税务查账补税罚税如何做账？
  • 6、装饰行业，连锁酒店行业的账务有何特色？
  • 7、所得税汇算清缴的如何进行？
  •  

    • 主管班

  • 对于财务人员，除了做账，还得清楚老板关心的是什么？会计人员如何自保？错弊账如何看出来？看出来了如何调整？企业有哪些财务制度，制度的出台与执行，特色行业的账务有何差异？会看报表吗？会分析利用数据吗？企业内部有哪些风险？如何制定实施有效的内控流程？如何在日常经营中做好税务查账的应对？如何在合理合法的情况下降低纳税成本？
  •  

    • 护航班

  • 拥有财务核算与财务管理实战能力，掌握大商业、 大工业等各类行业全盘账，掌握手工，财务软件，EXCEL做全盘帐，具备进入大中小企业会计岗位的实战能力，完全胜任企业财务主管，快速成为财税管理人 才。就业，换岗，加薪，晋升全搞定。
